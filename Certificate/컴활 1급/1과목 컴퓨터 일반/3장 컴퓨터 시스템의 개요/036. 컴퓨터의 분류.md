## 036. 컴퓨터의 분류

### 1. 컴퓨터 분류의 개요

컴퓨터는 다음과 같은 기준에 따라 분류할 수 있습니다.

- **처리 능력**에 따른 분류
- **취급하는 데이터의 형태**에 따른 분류
- **사용 목적**에 따른 분류

---

### 2. 처리 능력에 따른 분류

#### 슈퍼 컴퓨터(Super Computer, 초대형 컴퓨터)

![alt text](./img/슈퍼%20컴퓨터.png)

- 매우 높은 정밀도로 정확한 계산을 수행할 수 있습니다.
- 슈퍼컴퓨터의 속도를 측정하는 단위는 **플롭스(FLOPS)** 입니다.
- 인공위성 제어, 일기 예보, 시뮬레이션 처리, 초정밀 과학기술 연구, 지형 분석, 우주항공 산업 등의 특수 분야에 사용됩니다.

#### 메인프레임(Main Frame, 대형 컴퓨터)

![alt text](./img/메인%20프레임%20컴퓨터.png)

- 수백 명 이상의 사용자가 동시에 사용할 수 있는 대형 컴퓨터입니다.
- 은행, 병원, 정부 기관 등과 같이 대규모 데이터를 안정적으로 처리해야 하는 곳에서 사용됩니다.

#### 미니 컴퓨터(Mini Computer, 중형 컴퓨터)

![alt text](./img/미니%20컴퓨터.png)

- 중규모 시스템에 적합한 컴퓨터입니다.
- 학교, 연구소 등의 업무 처리나 과학기술 계산에 사용됩니다.

#### 마이크로 컴퓨터(Micro Computer, 소형 컴퓨터)

- **마이크로프로세서(MPU)** 를 CPU로 사용하는 컴퓨터입니다.
- 네트워크 환경에서는 주로 **클라이언트 역할**을 수행합니다.

---

### 3. 마이크로 컴퓨터의 종류

마이크로 컴퓨터에는 다음과 같은 종류가 있습니다.

- 데스크톱 컴퓨터
- 노트북 컴퓨터
- 태블릿 PC
- 스마트폰
- 워크스테이션

---

### 4. 데이터 취급에 따른 분류

컴퓨터를 데이터 취급 방식에 따라 분류한다는 것은, 컴퓨터가 처리하는 데이터의 형태인 **디지털형**, **아날로그형**, **혼합형**을 기준으로 나누는 것을 의미합니다.

#### 디지털 컴퓨터(Digital Computer, 범용 컴퓨터)

- 문자나 숫자로 표현되는 **비연속적 데이터(디지털형)** 를 처리하는 컴퓨터입니다.
- 일반적으로 우리가 사용하는 대부분의 컴퓨터가 여기에 해당합니다.
- **셀 수 있는 자료**를 취급합니다.

#### 아날로그 컴퓨터(Analog Computer, 전용 컴퓨터)

- 온도, 전류, 속도 등과 같이 **연속적으로 변화하는 데이터(아날로그형)** 를 처리하는 컴퓨터입니다.
- 특수 목적용 컴퓨터로 사용됩니다.
- **셀 수 없는 자료**를 취급합니다.

#### 하이브리드 컴퓨터(Hybrid Computer)

- 디지털 컴퓨터와 아날로그 컴퓨터의 장점을 결합하여 만든 컴퓨터입니다.

> **디지털형과 아날로그형**
>
> 디지털형은 결과를 이산적(비연속적으로 구분된)인 숫자나 문자로 표시하는 방식입니다.  
> 반면 아날로그형은 연속적인 값으로 표시하는 방식입니다.
>
> 예를 들어, 시간이 숫자로 표시되는 시계는 **디지털형**이고,  
> 시침과 분침으로 시간을 나타내는 일반 시계는 **아날로그형**입니다.

---

### 5. 디지털 컴퓨터와 아날로그 컴퓨터의 비교

| 항목 | 디지털 컴퓨터 | 아날로그 컴퓨터 |
| --- | --- | --- |
| 입력 형태 | 숫자, 문자 | 전류, 전압, 온도, 속도 |
| 출력 형태 | 숫자, 문자 | 곡선, 그래프 |
| 연산 형식 | 산술 연산, 논리 연산 | 미적분 연산 |
| 연산 속도 | 느림 | 빠름 |
| 구성 회로 | 논리 회로 | 증폭 회로 |
| 프로그래밍 | 필요함 | 필요하지 않음 |
| 정밀도 | 필요한 한도까지 가능 | 제한적 |
| 기억 기능 | 있음 | 없음 |
| 적용성 | 범용 | 특수 목적용 |
| 가격 | 고가 | 저가 |

---

### 6. 사용 용도에 따른 컴퓨터의 분류

컴퓨터는 사용 목적에 따라 **범용 컴퓨터**와 **전용 컴퓨터**로 분류할 수 있습니다.

#### 범용 컴퓨터

- 여러 분야에서 다양한 용도로 사용하기 위해 제작된 컴퓨터입니다.
- 일반적으로 **디지털 컴퓨터**가 여기에 해당합니다.

#### 전용 컴퓨터

- 특정한 목적에만 사용하기 위해 제작된 컴퓨터입니다.
- 자동 제어 시스템, 항공기술, 산업용 제어 분야 등에 사용됩니다.
- 일반적으로 **아날로그 컴퓨터**가 여기에 해당합니다.
