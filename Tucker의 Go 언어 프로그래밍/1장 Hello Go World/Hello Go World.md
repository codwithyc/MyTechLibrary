## Hello Go Wolrd

프로그래밍을 배우면 가장 먼저 해보는 것이 바로 Hello World 출력하기입니다.
이 과정은 아주 간단하지만, 앞으로 Go 언어로 코드를 작성하고 실행하는 흐름을 익히는 첫 단계입니다.

이번 글에서는 Go 설치 방법은 생략하고,
Go 파일을 만들고 Hello World를 출력하는 가장 기본적인 방법만 아주 쉽게 알아보겠습니다.

### Go로 가장 먼저 해보는 출력

이번 챕터에서 만들 파일은 아래 문장을 출력하는 것입니다.

```
Hello Go World
```

처음에는 이 한 줄을 출력하는 것도 충분히 의미가 있습니다.
왜냐하면 이 과정 안에 다음과 같은 기본 흐름이 들어 있기 때문입니다.

Go 파일 만들기
코드 작성하기
코드 실행하기
결과 확인하기

즉, 아주 짧은 코드지만 프로그래밍의 시작 흐름 전체를 경험하는 연습이라고 볼 수 있습니다.

### 테스트 코드 작성
먼저 `hello.go`라는 이름의 파일을 하나 만들어 주시면 됩니다.

그리고 아래 코드를 작성해주세요.

```go
package main

import "fmt"

func main() {
	fmt.Println("Hello World")
}
```

### 코드 하나씩 보기

1. package main

go 파일은 어던 묶음에 속하는지 먼저 적습니다.  
여기서 `main`은 프로그램을 실행할 수 있는 특별한 패키지 이름입니다.

간단하게 설명하자면 `package main` -> "이 파일은 실행할 프로그램입니다." 라고 알려주는 표시입니다.

2. import "fmt"

`import`는 필요한 도구를 가져오는 것입니다.  
여기서 `fmt`는 글자를 화면에 출력할 때 사용하는 Go의 기본 도구입니다.

3. func main()

`func` 는 함수를 만든다는 뜻입니다.  
그중에서 `main()`은 프로그램이 시작될 때 가장 먼저 실행되는 특별한 함수입니다. 

4. fmt.Println("Hello Go World")
이 부분이 실제로 화면에 글자를 출력하는 코드입니다.

- fmt → 출력 도구
- Println → 글자를 출력하는 기능
- "Hello World" → 실제로 출력할 내용

### 실행 방법

1. 터미널로 파일 위치 이동하기 
예를 들어 파일이 바탕화면의 `hello-go` 폴더 안에 있다면

```
cd ~/Desktop/hello-go
```

즉, `hello.go`가 있는 폴더로 들어가면 됩니다.

2. 실행하기
아래 명령어를 입력합니다.

```
go run hello.go
```

3. 해당 명령어가 무슨 뜻인가?

- go → Go 도구 실행
- run → 코드를 바로 실행
- main.go → 실행할 파일 이름

### 마무리

처음에는 코드를 완벽히 외우려고 하기보다,
