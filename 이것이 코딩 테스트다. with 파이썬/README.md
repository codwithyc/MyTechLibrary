## 이것이 취업을 위한 코딩 테스트다 with 파이썬

![alt text](./img/book_cover.png)

> 취업 준비를 하면서 코딩 테스트는 정말 면접까지 가기 위한 기본 코스라고 생각합니다.  
> 이 책을 통해서 코딩 테스트를 완벽하게 준비하고 저의 문제 풀이 레포지토리인 <a href="https://github.com/codwithyc/Algorithms">Algorithms</a> 의   
> 목표인 백준 루비 티어가 될때까지의 여정이 담긴 README입니다.

> 해당 책은 파이썬을 기반으로 다루어지고 있지만 저는 저의 메인 언어인 Java를 통해서 코딩테스트를 준비하려고 합니다.
