## if문

### if문이란?
---
프로그램은 항상 위에서 아래로만 실행되지 않습니다.  
어떤 조건이 맞을 때만 실행해야 하는 코드가 있고, 조건에 따라 다른 결과를 만들어야 할 때도 있습니다.

이때 사용하는 문법이 `if`문입니다.

```go
if 조건식 {
	조건식이 true일 때 실행할 코드
}
```

조건식의 결과는 반드시 `bool`이어야 합니다.  
즉 `true` 또는 `false`가 되는 식이어야 합니다.

```go
age := 20

if age >= 19 {
	fmt.Println("성인입니다.")
}
```

Go의 `if`문은 조건식에 괄호를 쓰지 않습니다.  
대신 중괄호 `{}`는 반드시 필요합니다.

```go
// Go 스타일
if age >= 19 {
	fmt.Println("성인입니다.")
}
```

### if문 기본 사용법
---
`if`만 사용할 수도 있고, 조건이 맞지 않을 때 실행할 코드를 `else`로 작성할 수도 있습니다.

```go
score := 80

if score >= 60 {
	fmt.Println("합격입니다.")
} else {
	fmt.Println("불합격입니다.")
}
```

조건이 여러 개라면 `else if`를 사용합니다.

```go
score := 85

if score >= 90 {
	fmt.Println("A")
} else if score >= 80 {
	fmt.Println("B")
} else if score >= 70 {
	fmt.Println("C")
} else {
	fmt.Println("D")
}
```

위 코드는 위에서부터 조건을 검사하다가 처음으로 `true`가 되는 블록만 실행합니다.  
한 번 실행할 블록을 찾으면 나머지 조건은 검사하지 않습니다.

### 그리고 &&, 또는 ||
---
조건을 여러 개 조합할 때는 논리 연산자를 사용합니다.

- `&&`
    양쪽 조건이 모두 참이어야 합니다.
- `||`
    양쪽 조건 중 하나만 참이어도 됩니다.
- `!`
    조건을 반대로 뒤집습니다.

```go
age := 22
hasTicket := true

if age >= 19 && hasTicket {
	fmt.Println("입장할 수 있습니다.")
}
```

`&&`는 두 조건이 모두 맞아야 하므로, 나이 조건과 티켓 조건이 모두 참일 때만 실행됩니다.

```go
isWeekend := true
isHoliday := false

if isWeekend || isHoliday {
	fmt.Println("쉬는 날입니다.")
}
```

`||`는 둘 중 하나만 참이어도 실행됩니다.

조건식이 복잡해지면 변수로 나누는 것이 더 읽기 좋습니다.

```go
isAdult := age >= 19
canEnter := isAdult && hasTicket

if canEnter {
	fmt.Println("입장할 수 있습니다.")
}
```

좋은 조건문은 읽는 사람이 "왜 이 코드가 실행되는지" 바로 이해할 수 있어야 합니다.

### 중첩 if
---
`if`문 안에 다시 `if`문을 넣을 수 있습니다.  
이것을 중첩 if문이라고 합니다.

```go
age := 20
hasTicket := true

if age >= 19 {
	if hasTicket {
		fmt.Println("입장할 수 있습니다.")
	}
}
```

하지만 중첩이 깊어질수록 코드는 읽기 어려워집니다.  
가능하면 논리 연산자로 조건을 합치거나, 먼저 제외할 조건을 처리하는 방식이 좋습니다.

```go
if age < 19 {
	fmt.Println("나이 제한으로 입장할 수 없습니다.")
	return
}

if !hasTicket {
	fmt.Println("티켓이 없어 입장할 수 없습니다.")
	return
}

fmt.Println("입장할 수 있습니다.")
```

이런 방식은 조건을 하나씩 통과하는 흐름이 분명해집니다.  
특히 함수 안에서는 잘못된 조건을 먼저 처리하고 빠져나오는 방식이 자주 사용됩니다.

### if 초기문; 조건문
---
Go의 `if`문에는 조건을 검사하기 전에 짧은 문장을 하나 실행할 수 있는 문법이 있습니다.

```go
if 초기문; 조건문 {
	실행할 코드
}
```

예를 들어 나눗셈 함수를 호출하고, 성공 여부를 바로 검사할 수 있습니다.

```go
if result, ok := divide(10, 2); ok {
	fmt.Println("결과:", result)
} else {
	fmt.Println("나눗셈에 실패했습니다.")
}
```

여기서 `result`와 `ok`는 `if`문 안에서만 사용할 수 있습니다.  
즉 변수의 범위가 좁아집니다.

이 문법은 다음과 같은 장점이 있습니다.

- 조건 검사에만 필요한 변수를 짧게 사용할 수 있습니다.
- 불필요하게 바깥 범위에 변수를 남기지 않습니다.
- 에러 처리나 성공 여부 확인을 간결하게 작성할 수 있습니다.

처음에는 낯설 수 있지만, Go 코드에서 자주 보이는 형태입니다.

### 핵심 요약
---
- `if`문은 조건에 따라 실행 흐름을 나누는 문법입니다.
- 조건식의 결과는 반드시 `bool`이어야 합니다.
- Go에서는 조건식에 괄호를 쓰지 않고, 중괄호는 반드시 사용합니다.
- `else if`, `else`를 사용해 여러 흐름을 표현할 수 있습니다.
- `&&`, `||`, `!`로 조건을 조합할 수 있습니다.
- `if 초기문; 조건문` 형태를 사용하면 조건문 안에서만 필요한 변수를 짧게 선언할 수 있습니다.
